# DeepResearch

一个基于AI的智能研究助手，能够自动生成研究计划、执行搜索查询并生成综合研究报告。

## 环境配置

1. 后端环境
   - 安装Python依赖: `cd backend && pip install -r requirements.txt`
   - 配置环境变量: 复制`.env.example`为`.env`并填入自己的API密钥

2. 前端环境
   - 安装依赖: `cd frontend && npm install`
   - 启动开发服务器: `npm start`

## 项目结构

- `/frontend`: React前端应用
- `/backend`: Flask后端API服务
